开始使用直播推流

你还没有配置 SRS 流媒体服务器。前往设置填写服务器地址后，即可自动生成专属推流链接。

推流地址
rtmp://your-srs-server/live/nk_dev_000_xxxxxxxx
Stream Key： nk_dev_000_xxxxxxxx
OBS 快速联动
OBS WebSocket 未连接

确保 OBS 已安装并开启 WebSocket 服务（OBS → 工具 → obs-websocket 设置 → 启用）。

或者

无法使用 WebSocket？导出配置文件后在 OBS 中手动导入（仅需操作一次）。

如何使用 OBS 推流？
  1. 确保 SRS 服务器已启动（可在设置页「测试 SRS 连通性」中验证）
  2. 点击「一键配置 OBS 推流」，等待提示「OBS 已就绪」
  3. 在 OBS 中点击「开始推流」，本页面状态将自动更新为「直播中」
  4. 如 OBS WebSocket 不可用，导出配置文件后在 OBS「文件 → 导入」中载入

轻量级 Windows 设备监控桌面端 · Powered by Electron

欢迎使用 Neko Status

首次使用请配置服务器地址以启用在线功能。
连接成功后即可登录或注册，设备密钥将在登录后自动生成。

服务器已连接，是否登录？

登录后可同步账户设置、自动配置设备密钥，在多端间无缝管理。
你也可以跳过登录，直接在设置中填入从网页复制的设备密钥使用。

密钥安全警告

检测到密钥安全事件，请确认以下信息。

密钥变更确认

您即将更改设备密钥。如果该密钥已绑定到其他设备，该设备的上报数据将被清除。

如果此密钥是您之前在本机使用的，且您只是在本地删除后重新填入，可以安全继续。否则请确认您了解数据清除的风险。
