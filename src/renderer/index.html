Neko Status
轻量级 Windows 设备监控桌面端 · Powered by Electron
Neko Status 是一款基于 Electron 构建的本地设备状态监控客户端，实时采集 CPU、内存、截图、进程等系统数据，通过玻璃拟态 UI 优雅呈现。支持云端同步、开机自启与远程监控，是你的设备向世界实时汇报的小助手喵～
