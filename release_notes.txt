## 新功能

- 新增 SRS 直播推流模块：支持配置 SRS 流媒体服务器地址、RTMP 端口、应用名及 HTTP API 端口
- 推流页面：自动生成专属推流地址（含 Stream Key），支持一键复制
- Stream Key 管理：支持重新生成推流密钥
- 推流状态实时监控：自动轮询 SRS API，展示当前推流在线/离线状态
- OBS WebSocket 联动：支持测试 OBS WebSocket 连接、一键推送推流配置到 OBS
- OBS 配置文件导出：无法使用 WebSocket 时可导出服务配置文件手动导入 OBS
- 侧边栏新增「直播推流」导航入口

## 改进

- 全部新增 UI 组件严格遵循项目设计规范（glass-input、action-btn、CSS 变量体系）
- 推流配置集成至设置页，与现有设置体验一致
