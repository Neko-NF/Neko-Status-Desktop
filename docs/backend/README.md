## 当前阶段说明（Beta）

**当前处于 Beta 迭代阶段，后端实现优先级如下：**

```
P0（阻塞性功能，必须先做）
├── 更新系统（见文档 05）—— 客户端必须能收到更新并完成安装
├── 设备认证 + API Key 鉴权（见文档 06）
└── 核心上报接口（见文档 02：设备状态、截图上传）

P1（Beta 阶段内完成）
├── WebSocket 实时推送
├── 历史记录查询
└── 健康度统计

P2（后续版本）
├── 多设备管理
├── 告警通知
└── 配置云同步
```

## 快速链接

- 客户端服务器地址配置：`设置 → 数据与上传 → 服务器地址`，默认 `https://api.koirin.com/neko`
- 本地测试模式：客户端支持切换到 `http://localhost:8080`，无需 API Key
- WebSocket 端点：`wss://{host}/neko/ws`
